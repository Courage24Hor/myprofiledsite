Coffee Consumption & Spending Insights

This page provides a concise walk-through of my data analysis project exploring coffee consumption behavior, demographic patterns, and spending trends based on survey data.

Introduction

This project analyzes coffee consumption patterns across age groups and genders using survey responses from over 2,700 individuals. The goal was to uncover behavioral trends, spending habits, and potential factors influencing coffee use.

Data Collection & Processing

The dataset consisted of 2,700+ rows with over 110 columns. Cleaning and transformation included handling null values, standardizing demographic fields, grouping age categories, encoding consumption frequency, and preparing measures for visualization.

All analysis and visualization were conducted using Power BI, with additional transformation done using Power Query.
